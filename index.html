Lavagna is an open-source issue/project management tool designed for small teams. Lightweight, pure Java, easy to install, easy to use.

View on github Download latest release: 1.0.6.1

Easy on the eyes, realtime UI

Lavagna strives for simple user interface and experience. Using a combination of sock.js and stomp, the UI is updated in realtime, even when websocket is not available.

Keep your projects under control

Multiple projects, Multiple boards. A nice dashboard view gives you a quick glance at the status of your work.

Milestone overview and project statistics are just one click away.

With the ics feed you can have your tasks in your preferred calendar application.

Powerful search

Lavagna search engine let you refine your query by using all the metadata available, plus full text search.

Customizable labels

Lavagna supports unlimited user defined typed label. Be it a simple label, a card or user reference, a list, Lavagna has them all.

Fine grained access control

Define global and per project permission for each single user.

Multiple login providers

Lavagna supports the most widespread authentication providers, such as OAuth (gitlab (gitlab.com and self hosted), github, google, bitbucket, twitter), Persona, and LDAP. No passwords are stored in the application.

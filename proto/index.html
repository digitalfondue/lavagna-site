Lavagna is an open-source issue/project management tool designed for small teams. Lightweight, pure java, easy to install and use.

Download latest release: 1.0-M1

Simple and realtime UI

Lavagna strive for a simple user interface and experience. Using sock.js + stomp the UI is updated in realtime even when websocket is not available.

Keep your projects under control

Lavagna is project based: under a project you can have multiple boards.

With a nice dashboard you can have a fast overview of the whole project or a single board. Additionally a specific milestone page is present for your planning.

Multi projects support with fine grained access control

For each project the administator can specify roles with different capabilities for each user.

Multiple login providers supported

Lavagna operate with a white list approach. The valid user must be added in the system. We delegate only the login part to a external provider so we don't have any password in the system

  • ldap
  • oauth: github/bitbucket/twitter/google
  • mozilla persona
  • test (use it only for testing purpose!)

Customizable per project labels

Lavagna support user defined typed label. For example you can define a "Priority" tag with a restricted set of value. The currently supported type of labels are:

  • empty label: no user defined value can be set
  • string label: the user can attach an additional text
  • number label: the user can attach a number
  • date label: the user can attach a date
  • user label: the user can attach a user to the label
  • card label: the user can reference a card
  • list label: the user can select from a predefined list of values

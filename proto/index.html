Lavagna is an open-source issue/project management tool designed for small teams. Lightweight, pure Java, easy to install, easy to use.

Download latest release: 1.0-M1

Easy on the eyes, realtime UI

Lavagna strive for a simple user interface and experience. Using a combination of sock.js and stomp, the UI is updated in realtime, even when websocket is not available.

Keep your projects under control

Multiple projects, Multiple boards. A nice dashboard view gives a quick glance at the status of you work.

Milestone overview and project statistics are just one click away.

Project documentation

Powerful search

Lavagna search engine let you refine your query by using all the metadata available and full text search.

Search documentation

Customizable labels

Lavagna supports unlimited user defined typed label. Be it a simple label, a card or user reference, a list, Lavagna has them all.

Labels documentation

Fine grained access control

Define global and per project permission for each single user.

Roles documentation

Multiple login providers

Lavagna supports the most widespread authentication providers, such as OAuth, Persona, and LDAP. No passwords are stored in the application.

Login Providers documentation

Roadmap

Below is a very high level roadmap of the feature we plan to implement. Things can change at any time, but it should give an idea of where the project is going.

Release 1.1

  • Infrastructure
    • Angular 1.4 or later
    • Security improvements (CSP)
  • General Features
    • Basic keyboard navigation
    • RCS commit integration
    • Task iCalendar
  • Project
    • Improved project administration UIs
    • Delete a project

Release 1.0

  • Basic features
    • Multiple Login support 1.0-M1
    • Export & Import 1.0-M1
    • MySQL support 1.0-M1
    • PosgreSQL support 1.0-M1
    • HSQLDB support 1.0-M1
    • Parameterized search 1.0-M1
    • E-Mail notifications 1.0-M1
  • Project
    • Project specific roles 1.0-M1
    • Milestones 1.0-M1
    • Statistics 1.0-M1
    • Typed labels 1.0-M1
    • Labels metadata 1.0-M2
  • Board
    • Bulk operations 1.0-M1
    • Archive/Backlog/Trash 1.0-M1
    • Column operations 1.0-M1
  • Card
    • Basic metadata such as Due Date, Assign/Watch, Milestones, Created/Modified 1.0-M1
    • Description 1.0-M1
    • Comments and file upload 1.0-M1
    • Action lists 1.0-M1
  • Documentation
    • Online documentation
    • Embedded documentation
